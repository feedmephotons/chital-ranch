# Chital Ranch - Project Overview

## Project Description
Professional website for Chital Ranch, a South Texas breeding operation specializing in chocolate fallow deer. The site showcases the herd, breeding program, and available stock for both breeders and hunting ranches.

## Technology Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Custom CSS
- **Email**: Resend API
- **Deployment**: Ready for Vercel/Netlify

## Site Structure

### Pages & Sections
1. **Home Page** (`/`)
   - Hero Section - Main headline with background image
   - Welcome Section - Introduction to Chital Ranch
   - Highlights - Three-column feature highlights
   - Quote Banner - Signature quote

2. **About Us Section**
   - Ranch history and breeding philosophy
   - Values and approach

3. **The Herd Section**
   - Breeding program details
   - Genetics information
   - Current sires and bloodlines

4. **For Sale Section**
   - Available animals (live cover, covered does, breeder bucks, yearlings)
   - Request availability form

5. **Visit the Ranch Section**
   - Appointment scheduling information
   - Schedule visit form

6. **Contact Section**
   - Contact information
   - Contact form
   - Location details
