# Chital Ranch - Design System

## Color Palette

### Primary Colors
```
Navy:     #1a2332  - Headers, primary text, professional elements
Charcoal: #2d3748  - Secondary text, borders
Gold:     #d4af37  - Primary CTA buttons, accents, highlights
```

### Neutral Colors
```
Cream:    #f7f5f0  - Page background
Tan:      #c9b896  - Supporting backgrounds
Stone:    #8b8378  - Muted text
```

### Accent Colors
```
Gold Light: #e6c960  - Hover states
Gold Dark:  #b8941f  - Active states
```

## Typography

### Font Families
- **Headings**: Georgia, Cambria, Times New Roman (Serif)
- **Body**: Inter, system-ui (Sans-serif)

### Font Sizes & Weights
```
H1: 3xl-6xl (48px-60px+) - Bold
H2: 2xl-4xl (36px-48px) - Bold
H3: xl-2xl (24px-30px) - Semibold
Body: base-lg (16px-18px) - Regular
Small: sm (14px) - Regular
```

## Spacing
- Section padding: 16-24 (64px-96px)
- Container max-width: 7xl (1280px)
- Grid gap: 6-8 (24px-32px)

## Components

### Buttons
1. **Primary Button** (`.btn-primary`)
   - Gold background (#d4af37)
   - White text
   - Hover: Darker gold (#b8941f)

2. **Secondary Button** (`.btn-secondary`)
   - Navy background (#1a2332)
   - White text
   - Hover: Charcoal (#2d3748)

3. **Outline Button** (`.btn-outline`)
   - Gold border
   - Gold text
   - Hover: Gold fill with white text

### Cards
- White/cream background
- Subtle shadow
- Rounded corners (8px)
- Padding: 6-8 (24px-32px)

### Forms
- Input fields: Border with charcoal, focus ring in gold
- Labels: Charcoal text, semibold
- Error states: Red border and text
- Success states: Green border and text

## Layout Patterns

### Section Container
- Max width: 1280px
- Horizontal padding: 4-8 (16px-32px)
- Vertical padding: 16-24 (64px-96px)

### Grid Layouts
- 1 column (mobile)
- 2 columns (tablet)
- 3 columns (desktop)

## Image Specifications

### Hero Image
- Dimensions: 1920x1080
- Format: JPG
- Subject: Texas ranch landscape with misty atmosphere

### Section Backgrounds
- Dimensions: 1920x1080
- Format: JPG
- Opacity: 10-30% overlay

### Gallery Images
- Dimensions: 800x600
- Format: JPG
- Subject: Chocolate fallow deer, ranch scenes

### Icons
- Dimensions: 256x256
- Format: PNG (transparent)
- Style: Simple, professional line art

## Accessibility

### WCAG 2.1 AA Compliance
- Color contrast ratio: 4.5:1 minimum
- Focus indicators: Visible on all interactive elements
- Alt text: All images
- ARIA labels: Forms and interactive elements
- Keyboard navigation: Full support

## Animation & Transitions
- Duration: 200ms (fast), 300ms (standard)
- Easing: ease-in-out
- Smooth scroll behavior
- Fade-in on scroll for sections
